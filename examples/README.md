# Example Applications

This directory contains example applications demonstrating how to instrument your services with OpenTelemetry and send telemetry data to ObserveX.

## Coming Soon

Example applications will be provided for:
- Node.js/Express
- Python/Flask
- Go/Gin
- Java/Spring Boot

Each example will demonstrate:
- Automatic instrumentation
- Manual instrumentation
- Custom metrics, logs, and traces
- Best practices for observability
